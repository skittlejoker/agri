### 2. Push Your Code

```bash
cd agriculture-marketplace
git init
git add .
git commit -m "Deploy to GitHub Pages"
git remote add origin https://github.com/YOUR_USERNAME/Agri_Market.io.git
git branch -M main
git push -u origin main
```

### 3. Enable GitHub Pages

1. Go to: `https://github.com/YOUR_USERNAME/Agri_Market.io/settings/pages`
2. Under **Source**, select **GitHub Actions**
3. Wait 2-3 minutes
4. Visit: `https://YOUR_USERNAME.github.io/Agri_Market.io/`
